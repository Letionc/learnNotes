# 对于学习者
一个存放**学习笔记**的网页。

你通常可以找到**很多资料**。

你可以从左下角**直接**点开**一则资料**，也可以从**输入框**搜索资料。

该网页使用了 **markdown** 编译器，但增加了一些**特点**：
- 设置特色**样式表**，便于抄在纸上。
- 点击 "Turn On" / "Turn Off" **方便背诵**。
